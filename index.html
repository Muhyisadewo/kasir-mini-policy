Kebijakan Privasi

Terakhir diperbarui: 28 April 2026

Aplikasi Kasir Mini Offline berkomitmen untuk melindungi privasi pengguna kami. Kebijakan ini menjelaskan bagaimana informasi Anda dikelola dalam aplikasi kami yang mengutamakan privasi dan keamanan loka.

1. Informasi yang Kami Kumpulkan

Aplikasi ini pada dasarnya adalah aplikasi OFFLINE.

  • Data Toko & Profil: Kami mengumpulkan informasi yang Anda masukkan secara sukarela seperti Nama Toko, Alamat, dan Nomor WhatsApp. Data ini digunakan murni untuk keperluan pencetakan struk dan identitas aplikasi.
  • Data Transaksi & Produk: Semua data produk, stok, dan riwayat penjualan disimpan secara lokal di perangkat Anda menggunakan database terenkripsi (Android Room dengan SQLCipher). Kami TIDAK mengirimkan data ini ke server kami.
  • Informasi Perangkat: Kami dapat mengumpulkan informasi teknis standar seperti versi sistem operasi untuk tujuan pemecahan masalah (crash reporting) secara anonim melalui Google Play Services.

2. Izin Aplikasi (Permissions)

Aplikasi ini memerlukan beberapa izin untuk berfungsi:

  • Kamera: Hanya digunakan untuk memindai barcode/QR code produk. Kami tidak mengambil foto atau video tanpa seizin Anda.
  • Biometrik: Digunakan untuk mengunci aplikasi. Proses otentikasi dilakukan oleh sistem Android; kami tidak memiliki akses ke data sidik jari atau wajah Anda.
  • Akses Google Drive: Jika Anda mengaktifkan fitur Backup, aplikasi akan meminta izin untuk menyimpan file cadangan ke Google Drive pribadi Anda. Data ini dikelola sepenuhnya oleh Anda sendiri.
  • Internet: Diperlukan untuk proses Login Google, Sinkronisasi Cloud, dan validasi status langganan Premium.

3. Penghapusan Data

Karena data disimpan secara lokal, Anda memiliki kendali penuh:

  • Menghapus Aplikasi: Menghapus aplikasi akan menghapus seluruh data transaksi di perangkat Anda secara permanen.
  • Data Cloud: Anda dapat menghapus file cadangan ('kasir_mini_v4.db') secara manual melalui akun Google Drive Anda kapan saja.

4. Keamanan Data

Karena semua data disimpan di dalam memori internal HP Anda, keamanan data tersebut bergantung pada keamanan perangkat Anda. Kami sangat menyarankan agar Anda:

  • Mengaktifkan fitur kunci aplikasi (Biometrik/PIN) yang telah kami sediakan.
  • Melakukan backup data secara rutin ke Google Drive untuk menghindari kehilangan data jika HP rusak atau hilang.
  • Menggunakan fitur keamanan perangkat seperti PIN layar kunci.

5. Pihak Ketiga

Aplikasi kami mungkin menggunakan layanan pihak ketiga yang dapat mengumpulkan informasi yang digunakan untuk mengidentifikasi Anda:

  • Google Play Services (autentikasi & crash reporting).
  • Google Drive API (sinkronisasi backup).
  • Google Play Billing (pembelian fitur premium).

6. Privasi Anak-anak

Layanan ini tidak ditujukan bagi siapa pun yang berusia di bawah 13 tahun. Kami tidak secara sengaja mengumpulkan informasi pribadi dari anak-anak di bawah 13 tahun.

7. Perubahan Kebijakan

Kami dapat memperbarui Kebijakan Privasi kami dari waktu ke waktu. Setiap perubahan akan diumumkan di halaman ini dan di dalam aplikasi.
