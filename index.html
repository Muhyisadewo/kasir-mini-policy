Kebijakan Privasi - Kasir Mini Offline

Terakhir Diperbarui: 24 Mei 2024

Hyee Code berkomitmen untuk melindungi privasi pengguna aplikasi Kasir Mini Offline. Kebijakan ini menjelaskan bagaimana informasi Anda dikelola.

1. Penyimpanan Data Offline

Aplikasi ini bekerja sepenuhnya secara offline. Semua data produk dan transaksi disimpan di memori internal perangkat Anda (Local Database). Kami tidak memiliki akses ke data tersebut.

2. Izin Kamera & Barcode

Izin kamera digunakan hanya untuk memindai barcode produk. Tidak ada data gambar yang dikirim keluar dari perangkat.

3. Keamanan Biometrik

Fitur sidik jari/wajah dikelola oleh sistem Android. Aplikasi tidak menyimpan data biometrik Anda.

4. Backup Google Drive

Jika Anda mengaktifkan fitur backup, data akan disimpan di akun Google Drive pribadi Anda sendiri melalui API resmi Google.

Penghapusan Akun dan Data (Account Deletion) Jika Anda ingin menghapus akun Google yang tertaut dengan aplikasi Kasir Mini Offline dan data terkait (seperti backup di Google Drive), silakan ikuti langkah berikut: 1. Buka aplikasi, masuk ke menu Setting. 2. Pilih opsi Logout atau Disconnect Google Drive. 3. Jika Anda ingin kami menghapus sisa data identitas yang kami akses melalui Google Sign-In, silakan hubungi kami melalui email di [Email Anda] dengan subjek "Permintaan Penghapusan Data". 4. Kami akan memproses penghapusan data identitas Anda dalam waktu 3-5 hari kerja. 5. Catatan: Data transaksi lokal di HP Anda akan terhapus secara otomatis jika Anda menghapus instalan aplikasi (Uninstall).
